# Générer un rapport PDF

Chaque visite validée peut être exportée en **PDF** pour archivage ou partage.

## Générer le PDF

1. Ouvrez le détail d'une visite (statut **Validé**)
2. Cliquez sur **Télécharger PDF** (ou l'icône PDF)
3. Le fichier se télécharge sur votre appareil

## Contenu du rapport

Le PDF inclut :

- **En-tête** : logo et nom de votre organisation (si personnalisé)
- **Informations générales** : date, parcelle, exploitation, conseiller
- **Observations** et **problèmes** détectés
- **Recommandations** avec produits et doses
- **Photos** avec les résultats d'analyse
- **Niveau d'urgence**

<Tip>
  Le PDF reprend automatiquement le **branding de votre organisation** (logo, couleurs) si votre administrateur l'a configuré dans les paramètres.
</Tip>

## Partager le rapport

Vous pouvez :
- **Envoyer par email** le PDF à l'agriculteur
- **L'imprimer** pour un dossier papier
- **L'archiver** dans votre système de gestion documentaire

L'agriculteur reçoit aussi un résumé par WhatsApp automatiquement lors de la validation (pas besoin de lui envoyer le PDF séparément, sauf s'il le demande).
