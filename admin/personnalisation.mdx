# Personnalisation

Les organisations avec un plan **Enterprise** peuvent personnaliser l'apparence de Topos.

## Options de personnalisation

| Élément | Description |
|---------|-------------|
| **Logo** | Votre logo apparaît dans la navigation et les rapports PDF |
| **Couleur principale** | La couleur dominante de l'interface |
| **Couleur secondaire** | Couleur d'accent |
| **Nom de l'application** | Remplace "Topos" par le nom de votre choix |
| **Slogan** | Texte affiché sous le logo |

## Configurer

<Steps>
  <Step title="Accéder aux paramètres">
    Allez dans **Paramètres** puis **Personnalisation**.
  </Step>

  <Step title="Uploader votre logo">
    Cliquez sur **Changer le logo** et uploadez votre fichier (PNG ou SVG recommandé, fond transparent).
  </Step>

  <Step title="Choisir les couleurs">
    Sélectionnez vos couleurs principale et secondaire via le sélecteur ou en entrant un code hexadécimal.
  </Step>

  <Step title="Nom et slogan">
    Renseignez le nom souhaité et un slogan optionnel.
  </Step>

  <Step title="Prévisualiser et enregistrer">
    Vérifiez le rendu dans l'aperçu, puis enregistrez.
  </Step>
</Steps>

## Impact

La personnalisation s'applique à :
- L'**interface** de l'application pour tous les membres de l'organisation
- Les **rapports PDF** (visites, bilans, registre phyto)
- Les **notifications WhatsApp** (nom de l'organisation)

<Note>
  La personnalisation n'est disponible qu'avec le plan **Enterprise**.
</Note>
