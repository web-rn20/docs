# Exporter le registre

Le registre phytosanitaire peut être exporté pour archivage, contrôle ou transmission.

## Formats disponibles

| Format | Usage |
|--------|-------|
| **PDF** | Document officiel avec en-tête, signature, prêt pour un contrôle |
| **Excel (.xlsx)** | Tableau formaté pour analyse ou intégration dans vos outils |
| **CSV** | Import dans un tableur ou logiciel tiers |
| **JSON** | Intégration technique avec d'autres systèmes |

## Comment exporter

1. Allez dans **Registre phyto**
2. Appliquez les **filtres** souhaités (exploitation, période, culture...)
3. Cliquez sur **Exporter**
4. Choisissez le format
5. Le fichier se télécharge sur votre appareil

## Contenu de l'export

L'export inclut :

- **En-tête** : nom de l'exploitation, SIRET, adresse, campagne
- **Tableau des traitements** : date, parcelle, culture, produit, AMM, dose, surface, opérateur
- **IFT calculé** par période
- **Signature** du responsable (si configurée)

<Tip>
  Pour un contrôle, exportez en **PDF** : c'est le format le plus adapté, avec mise en page officielle et en-tête de votre exploitation.
</Tip>
