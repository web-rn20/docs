# Créer une parcelle manuellement

Si votre parcelle n'apparaît pas dans le RPG ou si vous préférez la dessiner vous-même, vous pouvez la créer manuellement.

## Dessiner sur la carte

<Steps>
  <Step title="Ouvrir l'outil de dessin">
    Depuis la carte, cliquez sur **Ajouter une parcelle** puis **Dessiner**.
  </Step>

  <Step title="Tracer le contour">
    Cliquez sur la carte pour placer chaque **point** du contour de votre parcelle. Suivez les limites de la parcelle.

    - Chaque clic ajoute un point
    - Le contour se dessine au fur et à mesure
    - Cliquez sur le premier point pour **fermer** le polygone
  </Step>

  <Step title="Renseigner les informations">
    Complétez le formulaire :
    - **Nom** de la parcelle (ex: "Cabernet Sud")
    - **Culture** en cours
    - **Exploitation** à laquelle rattacher la parcelle
    - La **surface** est calculée automatiquement à partir du contour
  </Step>

  <Step title="Enregistrer">
    Cliquez sur **Enregistrer**. La parcelle apparaît immédiatement sur la carte.
  </Step>
</Steps>

## Modifier le contour

Pour ajuster le contour d'une parcelle existante :

1. Ouvrez le détail de la parcelle
2. Cliquez sur **Modifier le contour**
3. Déplacez les points existants ou ajoutez-en de nouveaux
4. Enregistrez

<Tip>
  Utilisez la **vue satellite** comme fond de carte pour tracer précisément les limites réelles de votre parcelle.
</Tip>
