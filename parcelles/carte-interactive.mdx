# La carte interactive

La carte est l'écran d'accueil de Topos. Elle vous donne une vue d'ensemble de vos parcelles et de l'activité terrain.

## Naviguer sur la carte

- **Zoom** : pincez (mobile) ou utilisez la molette (ordinateur)
- **Déplacer** : glissez avec un doigt (mobile) ou cliquez-glissez (ordinateur)
- **Centrer** : cliquez sur le bouton de géolocalisation pour centrer la carte sur votre position

## Ce qui est affiché

### Parcelles

Vos parcelles apparaissent comme des **polygones colorés** sur la carte. La couleur varie selon la culture :
- Vert : vigne
- Jaune : blé / céréales
- Orange : maïs
- Gris : autres cultures

Cliquez sur une parcelle pour voir :
- Son nom et sa culture
- Sa surface en hectares
- La date de la dernière visite
- Un accès rapide au briefing et à l'historique

### Marqueurs de visites

Les visites récentes sont indiquées par des **marqueurs** sur la carte. À faible zoom, les marqueurs proches sont **regroupés** automatiquement avec un compteur.

### Alertes

Si des problèmes (maladies, ravageurs) ont été détectés à proximité de vos parcelles, des **cercles d'alerte** s'affichent avec un code couleur selon la sévérité.

## Couches de fond

Changez le fond de carte selon vos besoins :

| Couche | Utilité |
|--------|---------|
| **Vue satellite** (IGN) | Voir le terrain réel, les haies, les bâtiments |
| **Cadastre** (IGN) | Identifier les limites parcellaires officielles |
| **RPG** | Voir les cultures déclarées du Registre Parcellaire Graphique |
| **Plan** (OpenStreetMap) | Vue classique avec les routes et noms de lieux |

Changez de couche via le **sélecteur de couches** en haut à droite de la carte.

## Actions depuis la carte

- **Cliquer sur une parcelle** : voir le détail
- **Bouton +** : lancer une nouvelle capture sur la parcelle sélectionnée
- **Double-cliquer** : zoomer sur la zone
