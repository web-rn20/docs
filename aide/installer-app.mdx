# Installer l'application

Petrichor est une **application web progressive (PWA)**. Vous pouvez l'installer sur votre écran d'accueil comme une application native, sans passer par l'App Store ou le Play Store.

## Sur Android (Chrome)

<Steps>
  <Step title="Ouvrir Petrichor">
    Ouvrez Petrichor dans **Chrome** sur votre téléphone Android.
  </Step>

  <Step title="Installer">
    Chrome vous proposera automatiquement d'installer l'application avec une **bannière en bas de l'écran**. Cliquez sur **Installer**.

    Si la bannière n'apparaît pas, cliquez sur les **3 points** en haut à droite puis **Installer l'application**.
  </Step>

  <Step title="C'est prêt">
    L'icône Petrichor apparaît sur votre écran d'accueil. Vous pouvez l'utiliser comme n'importe quelle application.
  </Step>
</Steps>

## Sur iPhone / iPad (Safari)

<Steps>
  <Step title="Ouvrir Petrichor">
    Ouvrez Petrichor dans **Safari** (obligatoire sur iOS, Chrome ne fonctionne pas pour l'installation PWA).
  </Step>

  <Step title="Partager">
    Appuyez sur le bouton **Partager** (carré avec une flèche vers le haut) en bas de l'écran.
  </Step>

  <Step title="Ajouter à l'écran d'accueil">
    Dans le menu de partage, sélectionnez **Sur l'écran d'accueil** (ou **Ajouter à l'écran d'accueil**).
  </Step>

  <Step title="Confirmer">
    Donnez un nom à l'application (ou gardez "Petrichor") et confirmez. L'icône apparaît sur votre écran d'accueil.
  </Step>
</Steps>

## Sur ordinateur (Chrome / Edge)

1. Ouvrez Petrichor dans Chrome ou Edge
2. Cliquez sur l'**icône d'installation** dans la barre d'adresse (un "+" dans un carré)
3. Confirmez l'installation

## Avantages de l'installation

- **Accès rapide** depuis l'écran d'accueil
- **Plein écran** sans barre de navigation du navigateur
- **Meilleur mode hors-ligne** : les données sont mieux mises en cache
- **Notifications** : recevez des alertes même quand l'app n'est pas ouverte
