# Questions fréquentes

<AccordionGroup>
  <Accordion title="L'IA se trompe dans la transcription, que faire ?">
    L'IA est très performante mais pas parfaite, surtout avec du bruit de fond ou un accent prononcé. Conseils :
    - Parlez **clairement** et **pas trop vite**
    - Évitez les environnements très bruyants (tracteur, vent fort)
    - Après le traitement IA, **relisez et corrigez** le compte-rendu avant de valider
    - Mentionnez les noms techniques clairement (noms de produits, maladies)
  </Accordion>

  <Accordion title="Puis-je utiliser Petrichor sans réseau ?">
    Oui ! Petrichor fonctionne en **mode hors-ligne** :
    - Vos enregistrements audio et photos sont stockés localement
    - Dès que le réseau revient, tout se synchronise automatiquement
    - Voir [Mode hors-ligne](/aide/mode-hors-ligne) pour plus de détails
  </Accordion>

  <Accordion title="Comment l'IA identifie les maladies sur les photos ?">
    L'IA utilise un modèle de vision avancé qui analyse l'image et recherche les symptômes visuels caractéristiques. Elle s'adapte automatiquement à la **culture** de la parcelle pour chercher les pathologies les plus probables (ex: mildiou et oïdium pour la vigne).
  </Accordion>

  <Accordion title="Mes données sont-elles sécurisées ?">
    Oui. Petrichor applique les mesures suivantes :
    - **Chiffrement** de toutes les communications (HTTPS)
    - **Isolation** des données par organisation (vos données ne sont jamais visibles par d'autres organisations)
    - **Chiffrement** des données sensibles au repos
    - Hébergement sur des serveurs **européens**
  </Accordion>

  <Accordion title="Combien de temps prend le traitement IA ?">
    En général moins de **10 secondes** pour une visite standard (audio + extraction). L'analyse de photos prend quelques secondes supplémentaires par photo.
  </Accordion>

  <Accordion title="Puis-je exporter mes données ?">
    Oui, vous pouvez exporter :
    - Le **registre phyto** en PDF, Excel, CSV ou JSON
    - Les **bilans de saison** en PDF
    - Les **comptes-rendus de visite** en PDF
  </Accordion>

  <Accordion title="Comment ajouter un agriculteur à mon organisation ?">
    En tant qu'admin, allez dans **Paramètres** puis **Membres**, et cliquez sur **Inviter**. Renseignez l'email de l'agriculteur et attribuez-lui le rôle **Agriculteur**.
  </Accordion>

  <Accordion title="L'agriculteur doit-il installer l'application ?">
    Non, l'agriculteur reçoit les comptes-rendus directement sur **WhatsApp**. Il peut aussi accéder à son portail via un navigateur web s'il le souhaite, sans installation.
  </Accordion>

  <Accordion title="Que se passe-t-il si je dépasse mon quota IA ?">
    Les traitements IA sont suspendus jusqu'au renouvellement mensuel. Vous pouvez passer à un plan supérieur pour augmenter votre quota. Vos données existantes restent accessibles.
  </Accordion>
</AccordionGroup>
