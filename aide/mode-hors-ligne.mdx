# Mode hors-ligne

Petrichor est conçu pour le terrain, où le réseau n'est pas toujours disponible. L'application fonctionne **même sans internet**.

## Ce qui fonctionne hors-ligne

| Fonctionnalité | Hors-ligne | Détail |
|---------------|-----------|--------|
| **Enregistrer un audio** | Oui | Stocké localement sur votre appareil |
| **Prendre des photos** | Oui | Stockées localement |
| **Consulter la carte** | Partiellement | Les tuiles déjà chargées sont en cache |
| **Voir les parcelles** | Partiellement | Les données en cache sont disponibles |
| **Lancer le traitement IA** | Non | Nécessite une connexion |
| **Valider une visite** | Non | Nécessite une connexion |

## Comment ça marche

### Pendant la capture (hors-ligne)

1. Vous enregistrez votre audio normalement
2. Vous prenez des photos normalement
3. Tout est **stocké dans la mémoire locale** de votre appareil
4. Un **indicateur** vous montre que vous êtes hors-ligne et que les données sont en attente

### Au retour du réseau

1. Petrichor détecte automatiquement le retour de la connexion
2. Les fichiers audio et photos sont **uploadés automatiquement**
3. La visite est créée avec toutes vos données
4. Vous recevez une **notification** quand la synchronisation est terminée

<Tip>
  Vous n'avez rien à faire : la synchronisation est **entièrement automatique**. Reprenez le traitement IA de retour au bureau, quand vous avez une bonne connexion.
</Tip>

## Conseils

- **Installez l'application** sur votre écran d'accueil pour de meilleures performances hors-ligne
- **Chargez la carte** de votre zone de travail avant de partir sur le terrain (naviguez dessus une première fois avec du réseau)
- Vérifiez que vous avez assez d'**espace de stockage** sur votre appareil
