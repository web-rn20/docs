# Connexion SSO

Si votre organisation a configuré le **SSO (Single Sign-On)**, vous pouvez vous connecter avec les identifiants de votre organisation (Active Directory, Google Workspace, etc.).

## Comment se connecter en SSO

<Steps>
  <Step title="Page de connexion">
    Sur la page de connexion, cliquez sur **Se connecter avec mon organisation**.
  </Step>

  <Step title="Entrer votre email">
    Renseignez votre email professionnel. Petrichor détecte automatiquement votre organisation et son provider SSO.
  </Step>

  <Step title="Authentification">
    Vous êtes redirigé vers la page de connexion de votre organisation (Microsoft, Google, etc.). Connectez-vous avec vos identifiants habituels.
  </Step>

  <Step title="Accès automatique">
    Après authentification, vous êtes automatiquement connecté à Petrichor avec les bonnes permissions.
  </Step>
</Steps>

## Première connexion SSO

Si c'est votre première connexion :
- Votre **compte Petrichor est créé automatiquement**
- Vous êtes ajouté à votre **organisation** avec le rôle par défaut (membre)
- Votre administrateur peut ensuite ajuster votre rôle si nécessaire

<Note>
  La connexion SSO est disponible uniquement pour les organisations avec un plan **Enterprise**. Contactez votre administrateur pour la configuration.
</Note>
