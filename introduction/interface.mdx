# Découvrir l'interface

## Navigation principale

L'interface de Petrichor est organisée autour de ces sections :

| Section | Icône | Description |
|---------|-------|-------------|
| **Carte** | Carte | Vue cartographique de vos parcelles et visites |
| **Visites** | Liste | Liste de toutes vos visites terrain |
| **Capture** | Micro | Enregistrer un audio ou prendre une photo sur le terrain |
| **Parcelles** | Champs | Gérer vos parcelles agricoles |
| **Exploitations** | Ferme | Gérer vos exploitations |
| **Registre phyto** | Registre | Journal des traitements phytosanitaires |
| **Bilans** | Graphique | Bilans de saison et indicateurs |
| **Paramètres** | Engrenage | Profil, organisation, abonnement |

## La carte (écran d'accueil)

L'écran d'accueil affiche une **carte interactive** avec :
- Vos **parcelles** dessinées en couleur selon la culture
- Les **marqueurs de visites** récentes
- Les **alertes** de maladies à proximité

**Actions rapides depuis la carte :**
- Cliquer sur une parcelle pour voir son détail et son historique
- Cliquer sur un marqueur pour ouvrir une visite
- Utiliser le bouton **+** pour lancer une nouvelle capture

## Palette de commandes

Appuyez sur **Cmd+K** (Mac) ou **Ctrl+K** (Windows) pour ouvrir la **palette de commandes**. Elle vous permet de :
- Rechercher une parcelle, une exploitation ou une visite
- Naviguer rapidement entre les sections
- Lancer des actions (nouvelle visite, nouveau traitement)

## Mode terrain (Fat Finger)

Quand vous êtes sur le terrain avec des gants ou les mains sales, activez le **mode terrain** :
- Les boutons deviennent **plus grands**
- Les zones de clic sont **élargies**
- L'interface est simplifiée pour une utilisation facile en conditions réelles

Activez-le depuis le menu en haut à droite ou via la palette de commandes.

## Thème sombre / clair

Petrichor propose un thème **sombre** (par défaut) et un thème **clair**. Changez-le dans **Paramètres** ou il s'adapte automatiquement à votre préférence système.
